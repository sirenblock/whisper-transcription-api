## ðŸ”§ Configuration

### Change Grid Layout

Edit `calc_grid_dimensions()` in `claude-orchestrator.sh`:

```bash
calc_grid_dimensions() {
    local count=$1
    local cols=4  # 4 columns instead of 3
    local rows=3  # 3 rows instead of 4
    echo "$cols $rows"
}
```

### Adjust Keep-Alive Frequency

Edit generated `keep-alive-auto.sh`:

```bash
sleep 3  # Change to 5 for less frequent
```

### Use Different Claude Model

Modify launch command in orchestrator:

```bash
set newWindow to do script "claude --model opus"
```

## ðŸš¨ Troubleshooting

### "No PROJECT.md found"
Create one with your project description first.

### "Failed to break down project"
- Add more detail to PROJECT.md
- Check `.claude-orchestrator/breakdown.json`
- Ensure requirements are clear

### Windows not processing
- Run `./keep-alive-auto.sh`
- Manually press Enter in stuck windows
- Check for permission prompts

### Wrong number of tasks
- Claude optimizes for 4-12 tasks automatically
- Add/remove detail in PROJECT.md to influence count

### Grid layout issues
- Modify `calc_grid_dimensions()` function
- Manually resize if needed
